# AI Image Classification Project

This project is a Flask-based web application that performs image classification using a Convolutional Neural Network (CNN).

Features:
- Upload image through web interface
- Predict object category
- Shows prediction confidence
- Custom trained dataset

Technologies Used:
- Python
- PyTorch
- Flask
- HTML/CSS

How to Run:
1. Install dependencies
2. Run train_model.py (to train model)
3. Run app.py
4. Open localhost:5000
